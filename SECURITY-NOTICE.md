# 보안 및 개인정보 주의

이 프로그램은 학교폭력 사안, 학생 이름, 보호자 연락처, 주소, 조치 이력 등 민감한 정보를 다룰 수 있습니다.

## 권장 원칙
- 실데이터 운영판은 공개 GitHub Pages에 그대로 올리지 않습니다.
- 공개 저장소에는 샘플 데이터만 사용합니다.
- 실제 업무판은 비공개 저장소, 내부망, 또는 설치형 데스크톱 앱으로 운영합니다.
- 백업 파일(JSON)은 암호화 저장 또는 접근통제된 폴더에 보관합니다.
- 기본 비밀번호 `1234`는 즉시 변경합니다.

## 현재 구조의 한계
현재 웹 버전은 브라우저 저장소(localStorage) 기반입니다.
민감정보를 장기간 운용하려면 로컬 파일 저장, SQLite, 암호화, 사용자 권한 관리가 추가되어야 합니다.
