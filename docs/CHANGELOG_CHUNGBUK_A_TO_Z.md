# 충북 A to Z 반영 변경사항

- 조치 결정 전 용어를 피해관련 학생 / 가해관련 학생으로 정비
- 즉시분리 여부, 미시행 예외사유 6종, 분리기간/장소 조건부 입력 반영
- 가해관련학생 긴급선도조치 2호 필수 확인 항목 추가
- 긴급조치 입력 구조를 피해관련학생 보호조치 / 가해관련학생 선도조치로 분리
- 전담기구 개최일 2주, 3주(+연기), 관계회복 숙려제도 4주 옵션 반영
- 충북교육청 A to Z 서식/공문 번호 체계 중심으로 메뉴 명칭 수정
- GitHub 업로드 시 줄바꿈 혼선을 줄이기 위해 `.gitattributes` 추가
