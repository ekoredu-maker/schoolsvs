# 학교폭력 업무 담당자용 업무 처리 보조 프로그램v5

이 패키지는 **충청북도교육청 『2026. 학교폭력 사안처리 A to Z』 기준**을 반영한 **GitHub 업로드용 구조**입니다.

## 반영 기준
- 조치 결정 전 용어는 **피해관련 학생 / 가해관련 학생**으로 통일
- 학교장 인지 후 **24시간 이내 즉시분리 방법 결정**
- **가해관련학생 긴급선도조치 2호** 필수 확인
- **주말 포함 48시간 이내** 교육(지원)청 보고
- 전담기구 개최일은 **2주 이내**, 필요 시 **+1주 연기(내부결재)**, 관계회복 숙려제도 운영 시 **최대 4주** 반영
- 서식/공문 명칭은 가능한 범위에서 **충북 A to Z 번호 체계**에 맞춤

## 포함 파일
- `index.html` : 메인 앱
- `manifest.webmanifest` : 설치형 웹앱(PWA) 정보
- `sw.js` : 오프라인 캐시용 서비스 워커
- `.github/workflows/pages.yml` : GitHub Pages 배포용 워크플로
- `.gitattributes` : 줄바꿈(LF) 정리용 설정
- `.nojekyll` : 정적 배포용 설정
- `assets/icons/*` : 아이콘
- `SECURITY-NOTICE.md` : 민감정보 운영 주의사항
- `docs/DEPLOY_GITHUB_PAGES.md` : Pages 배포 안내



## 주의
이 앱은 `localStorage` 기반이므로, **공개 GitHub Pages에서 실데이터를 운영하는 것은 권장하지 않습니다.**
공개 웹판은 데모 또는 비식별 샘플 데이터 용도로 사용하세요.
